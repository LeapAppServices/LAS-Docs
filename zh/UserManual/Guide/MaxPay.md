# MaxPay

MaxPay 是为移动端应用以及 PC 网页量身打造的下一代支付系统，通过一个 SDK 便可以同时支持移动端以及 PC 端网页的多种主流支付渠道。

## WebHook
为了便于客户系统或者第三方系统处理客户的交易信息， MaxPay 提供 Webhooks 功能，可以按照客户要求把特定的事件结果推送到指定的地址以便于客户做后续处理。[详见 WebHook 使用指南](具体链接)
