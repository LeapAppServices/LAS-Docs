# Cloud Data
在"开发者中心"模块中，我们可以管理所有的Cloud Data.

##Class列表

###新建Class
点击"＋新建Class"按钮，输入Class名，便可以完成Class的新建：

Pic

###删除Class
在Claas列表中选中要删除的Class，在右侧选择 More >> Del Class，确认即可。

Pic

###修改Class
选中Class后，您可以对此Class进行如下修改：

* 新建Column：添加Class属性
* 新建Row：添加一条数据
* 筛选表格数据显示：点击沙漏图标
